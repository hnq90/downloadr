flickr downloadr

flickr downloadr is a web based tool to display or download all photos and photography from a flickr set in small, medium, large or original size.

Directions: Simply enter a Set ID or a Set URL, pick a size and then Get or Save the images. Click here to see a preview.

Due to this being a quick project I haven't completed 'Save Images' in anything but a Chrome Browser.
